# 📊 Remnawave Live Monitor & Dynamic Dashboard

Красивый адаптивный дашборд для мониторинга нод и онлайн-пользователей **Remnawave** с виджетом погоды, часами и календарём. Оптимизирован под любые экраны: от вертикальных инфо-стоек до 4K-мониторов.

## ✨ Возможности

- **Мониторинг Remnawave в реальном времени:** активные ноды (CPU, RAM, Load Average, трафик, онлайн-пользователи) с авто-сортировкой и live-обновлением.
- **Список онлайн-пользователей** с эмуляцией по порогу активности (менее 30 секунд).
- **Виджет погоды** с поиском любого города и быстрым выбором городов России (сохраняется в `localStorage`).
- **Часы и календарь** с автоподстройкой часового пояса под выбранный город.
- **Полная адаптивность:** одна колонка на вертикальных экранах, две колонки на горизонтальных.

## 🚀 Быстрый старт

### Windows (в один клик)
1. Скачайте проект в любую папку (например, `C:\Dashboard`).
2. Настройте конфигурацию в `start_dashboard.py` (см. ниже).
3. Запустите **`run_dashboard.bat`** — скрипт сам установит зависимости (`fastapi`, `httpx`, `uvicorn`) и откроет дашборд в браузере.

### Linux / macOS
```bash
pip install fastapi httpx uvicorn
python start_dashboard.py
```
Откройте в браузере: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## ⚙️ Настройка (`start_dashboard.py`)

Найдите в начале файла блок конфигурации:

```python
REMNA_API_URL = "https://your-domain.com"
REMNA_TOKEN = "YOUR_TOKEN"
COOKIE_SECRET = ""
```

Заполните его **в зависимости от способа установки вашей панели Remnawave**:

### 🟢 Вариант А: Ручная установка (Docker / Bare Metal)
Заполните только **два** параметра. `COOKIE_SECRET` оставьте пустым (`""`).
```python
REMNA_API_URL = "https://panel.example.com"
REMNA_TOKEN = "eyJhbGciOiJI..."
COOKIE_SECRET = ""
```

### 🟣 Вариант Б: Установка через скрипт от E-geme (или аналоги)
Скрипты часто настраивают Nginx с дополнительной защитой через cookies. Заполните **все три** параметра.

#### 🍪 Как получить `COOKIE_SECRET`:
1. Откройте панель Remnawave в браузере и войдите в аккаунт.
2. Нажмите `F12` → вкладка **Network (Сеть)**.
3. Обновите страницу (`F5`) и кликните на любой запрос к вашему домену.
4. В **Request Headers** найдите строку `Cookie:` и скопируйте всё её содержимое в `COOKIE_SECRET`.

> ⚠️ **Важно:** куки могут "протухать" после выхода из панели в браузере. Для продакшена рекомендуется чистая установка без сторонних скриптов.

### Как это работает под капотом
Заголовки формируются динамически — если `COOKIE_SECRET` пустой, лишний `Cookie` заголовок не отправляется. Это защищает пользователей с ручной установкой от ошибок `400 Bad Request`:

```python
HEADERS = {"Authorization": f"Bearer {REMNA_TOKEN}"}
if COOKIE_SECRET:
    HEADERS["Cookie"] = COOKIE_SECRET
```
