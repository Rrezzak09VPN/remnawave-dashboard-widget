# Remnawave Live Monitor & Dynamic Dashboard Widget

Красивый, полностью адаптивный дашборд-виджет для мониторинга активных нод и онлайн-пользователей **Remnawave**, дополненный динамической погодой, точными часами и интерактивным календарем.

Интерфейс идеально оптимизирован под любые устройства: от вертикальных информационных стоек и планшетов до широкоформатных горизонтальных 4K-мониторов.

---

## 🚀 Основные возможности

1. **Мониторинг Remnawave (в реальном времени):**
   * Список активных нод (IP-адрес, CPU, нагрузка Load Average, RAM, трафик в GB и онлайн ядра).
   * Сортировка нод по количеству активных пользователей.
   * Список онлайн-пользователей с эмуляцией по порогу активности (менее 30 секунд).
   * Автоматическое обновление данных и плавающий индикатор последнего обновления с красивым анимированным CSS-спиннером в углу экрана.

2. **Динамический виджет погоды с поиском городов:**
   * **Поиск любого города:** Возможность ввести название любого города прямо в виджете (через встроенный поиск с поддержкой геокодинга API).
   * **Популярные города:** Быстрый выбор популярных городов России в один клик.
   * **Сохранение выбора:** Выбранный город сохраняется в локальной памяти браузера (`localStorage`) и восстанавливается при перезапуске.
   * **Отказоустойчивое гибридное проксирование:** Запросы погоды идут через бэкенд сервера, но в случае неполадок бесшовно переключаются на прямой клиентский запрос к Open-Meteo.

3. **Интерактивные часы с автоподстройкой часового пояса:**
   * Показывают точное локальное время выбранного города (автоматически синхронизируются с таймзоной погоды).
   * Выводят дату и день недели на русском языке.
   * Адаптивный размер шрифта (плавно подстраивается под экраны от мобильных до 4K).

4. **Профессиональный интерактивный календарь:**
   * Русский формат дней недели (Пн-Вс).
   * Выделение текущего дня.
   * Удобные кнопки перелистывания лет (`«` / `»`), месяцев (`‹` / `›`) и недель (`‹ нед` / `нед ›`).
   * Кнопка `"Сегодня"` для быстрого возврата.

5. **Полная адаптивность и "резиновая" верстка:**
   * **На вертикальных экранах (портретная ориентация):** Блоки располагаются друг под другом в один аккуратный столбец, блок онлайн-пользователей уходит вниз.
   * **На горизонтальных экранах (альбомная ориентация):** Блоки выстраиваются в две колонки, эффективно используя всё полезное пространство.

## 🛠️ Установка и запуск

### Быстрый запуск на Windows (в один клик):
1. Скачайте файлы проекта и поместите их в любую удобную папку (например, `C:\Dashboard`).
2. Откройте файл `start_dashboard.py` через текстовый редактор и пропишите параметры вашей инстанции (см. раздел [Настройка](#-настройка)).
3. Дважды кликните по файлу **`run_dashboard.bat`**.
   * *Скрипт сам проверит наличие Python, при необходимости автоматически установит все библиотеки (`fastapi`, `httpx`, `uvicorn`), запустит локальный веб-сервер и сам откроет страницу в вашем браузере по умолчанию.*

---

## ⚙️ Настройка

Откройте файл `start_dashboard.py` и в самом начале в блоке конфигурации пропишите адрес вашего API Remnawave и секретный токен:

```python
# --- КОНФИГУРАЦИЯ ---
REMNA_API_URL = "https://your-remnawave-instance.com" # Адрес вашей панели Remnawave
REMNA_TOKEN = "YOUR_REMN_API_TOKEN"                   # Ваш Bearer токен авторизации
```

---

## 🖥️ Ручной запуск (для любых ОС)

Если вы запускаете проект на Linux, macOS или вручную в консоли Windows, выполните следующие команды:

1. **Установите зависимости:**
   ```bash
   pip install fastapi httpx uvicorn
   ```

2. **Запустите сервер:**
   ```bash
   python start_dashboard.py
   ```

3. **Откройте страницу в браузере:**
   Перейдите по ссылке [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## 🔒 Безопасность и конфиденциальность

* В репозитории отсутствуют какие-либо персональные ключи доступа, реальные токены и адреса серверов.
* Все запросы геокодинга и погоды абсолютно конфиденциальны и не передают личные данные пользователя.
* Код полностью открыт и безопасен для развертывания в локальных сетях.
